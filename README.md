# AMS-Gemma4: Antibiotic Stewardship Clinical Decision Support

Offline AI assistant for antibiotic recommendations in Indian hospitals.

Built for the Kaggle Gemma 4 Good Hackathon - Health & Sciences Track.

**Features:**
- WHO AWaRe 2023 classification
- ICMR India resistance patterns
- Cockcroft-Gault renal dose adjustment
- Safety gate (rejects contraindicated drugs)
- Tamil language output
- Runs offline via Ollama + Gemma 4 E2B

**Author:** Prakash Raj K, Associate Professor of Pharmacy, SBV, Puducherry, India
